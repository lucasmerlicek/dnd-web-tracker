# Product Overview

D&D Character Tracker — a desktop application for managing Dungeons & Dragons 5th Edition characters during tabletop sessions.

Each character has their own self-contained tracker folder (e.g. `Tracker_Barian`, `Tracker_Madea`). The app provides:

- Character stats, skills, and ability scores display
- Attack rolls and damage calculations with advantage support
- Spell management with slot tracking, preparation, metamagic, and sorcery points
- Saving throws and death saves
- Inventory and coin management
- Session journal with character/place tracking
- Interactive world map viewer with categorized markers (artifacts, treasure, enemies, people, notes)
- Short rest / long rest mechanics
- HP tracking with damage and healing input

The UI uses a dark fantasy aesthetic with custom background images and UI box textures per screen. The app targets Windows and adapts to display DPI scaling. Each tracker is tailored to a specific D&D character with class-specific features (e.g. sorcery points for sorcerers, bladesong for wizards).

The workspace also contains reference PDFs (D&D 5E rulebooks, character sheets), character backstory text files, inspiration images, and world maps.
